## Key Findings

### What Works
- Theme + Trend + Catalyst alignment
- Tech stocks (AMD, NVDA, PLTR)
- $1-6 options, 3-7% OTM
- Monday/Wednesday entries
- Overnight puts (63.6% win rate)

### What Doesn't Work
- Lottery tickets (<$1 options)
- Unfamiliar sectors
- Friday entries
- Fighting the trend
